## API 


| 라우트        | 메소드 | 설명              | 구현여부 |
| ------------- | ------ | ----------------- | -------- |
| /post         | POST   | 알고리즘 작성     |          |
| /post/{idx}   | GET    | 알고리즘 상세보기 |          |
| /post/{idx}   | PUT    | 알고리즘 수정     |          |
| /post/{idx}   | DELETE | 알고리즘 삭제     |          |
| /users        | POST   | 회원가입          | O        |
| /users        | GET    | 마이페이지        | O        |
| /users/signin | POST   | 로그인            | O        |
| /users/logout | GET    | 로그아웃          |          |
| /users        | DELETE | 회원탈퇴          |          |
| /users        | PUT    | 정보수정          |          |
